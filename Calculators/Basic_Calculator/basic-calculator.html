Frequently Asked Questions

How do memory functions work?

MC clears memory, MR recalls stored number, M+ adds current number to memory, M- subtracts current number from memory.

Can I use keyboard?

Yes, you can use number keys, operators (+, -, *, /), Enter for equals, and Escape for clear.
